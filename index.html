UMF

Users - Messages - Forums -- This is an advance package of A3M that adds to its basis functionality private messages and simple forums to kickstart the development of web apps.

Download Download View UMF on GitHub

Installation

  1. Copy all files into your web root
  2. Import the database in the database.sql into your MySQL database
  3. Adjust settings in config/database.php
  4. Adjust settings in folder config/umf/ to your liking
  5. Adjust the .htaccess file
  6. You should be all set!

UMF came from merging of

  • A3M
  • Mahana Messaging Library
  • CIBB

Dependencies

  • PHP 5.3
  • CURL
  • DOM or domxml
  • GMP or Bcmatch
  • MySQL
